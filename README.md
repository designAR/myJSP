# JSP회원가입, 게시판
AR삽질러 - https://arang95.tistory.com/64

나혼자 해보는 JSP WebProject

1. 기능
 - 메인
 - 회원가입
     아이디(중복체크), 비밀번호, 비밀번호확인, 이름, 전화번호, 나이, 성별, 이메일, 우편번호(찾기), 프로필(찾기)
 - 로그인
     회원관리(회원수정, 삭제, 조회)
 - 게시판
     글목록, 답변, 수정, 삭제, 페이지이동
